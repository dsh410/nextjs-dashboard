## Acme Dashboard Project

Welcome! This is a dashboard application I built by following the official Next.js App Router Course. The project showcases modern React (with Next.js) development practices, including:

- Next.js App Router and file-based routing
- Full-stack capabilities: server actions, API routes, and data fetching
- TypeScript for safe, maintainable code
- Tailwind CSS for rapid, utility-first styling
- Zod for robust form validation
- NextAuth for authentication
- PostgreSQL integration for real database operations

### What I Did

I followed a comprehensive tutorial from Next.js to build this dashboard from scratch, learning and applying core concepts such as:

- Setting up and structuring a scalable Next.js project
- Creating reusable UI components (forms, tables, buttons)
- Implementing CRUD operations (create, update, delete invoices)
- Server-side and client-side data validation
- Responsive design for desktop and mobile
- Managing authentication flows and secure data handling

Throughout this process, I gained hands-on experience building a real-world app, understanding how to connect frontend and backend code using Next.js features, and writing clean, production-ready code.
